# AWS Professional Notes 📚☁️

A structured, professional-grade collection of notes, cheatsheets, code
snippets, and real-world project walkthroughs for learning and mastering
**Amazon Web Services (AWS)** — from fundamentals to advanced
architecture patterns.

This repository is organized so you can follow it top-to-bottom as a
learning path, or jump directly to the service/topic you need as a
reference.

## 🚀 How to use this repo

1. **New to AWS?** Start at `00-prerequisites/` and move sequentially
   through the numbered folders.
2. **Studying for a certification?** Jump to
   `19-resources-cheatsheets/interview-questions.md` and
   `15-architecture-patterns-and-best-practices/`.
3. **Need a working example?** Check `17-real-world-projects/` for
   guided, end-to-end builds, and `18-code-snippets/` for reusable
   templates and scripts.
4. **Looking for a specific service?** Use the table above or your
   editor's file search to jump straight to it.

## 🛠️ Prerequisites

- An AWS account (Free Tier is enough for most examples)
- [AWS CLI v2](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html) installed and configured
- Basic familiarity with the command line (see `00-prerequisites/04-linux-and-command-line-basics.md`)
